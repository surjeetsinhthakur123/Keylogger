# Keystroke Logging Demonstration (GUI)

## Overview
A Python-based desktop application that demonstrates how keyboard input events are captured and processed using a graphical user interface. The project is intended **only for educational purposes with user consent**.

## Features
- Premium Tkinter-based GUI  
- Start and stop monitoring controls  
- Real-time keystroke display  
- Logs stored in **JSON** and **TXT** formats  
- Timestamped session-based logging  

## Technologies
- Python  
- Tkinter  
- pynput  
- JSON  

## How to Run
```bash
pip install pynput
python Keylogger.py
```

## Notes
- Runs **only on local machines**
- Not supported on cloud platforms
- For academic demonstration purposes only
